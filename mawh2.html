Make-A-Wish Hawaii Internship


Cheers! 2026 Event Materials


Background

The print materials that I created for Make-A-Wish Hawaii's 2026 Cheers event. This collection, which takes on MAWH's more sophisticated/formal branding for Cheers!, consists of table cards, a large sponsor sign, menu signs, and wine tags.

All assets were assembled in a combination of Adobe Photoshop, Illustrator, and InDesign.

Outcome

Cheers! Table Cards
These table cards were assembled via pre-existing template from past Cheers! events in Adobe Photoshop, using updated photos and Wish Kid bios/quotes and rearranging assets as necessary. 





Cheers! X Sponsor Banner
This sponsor banner, which highlights all the sponsors that contributed to this event by donation size, was assembled in Adobe Illustrator utilizing sponsor logos and other design elements from MAWH's asset library.




Cheers! Wine Tags
These wine wall tags, highlighting MAWH's Young Leaders Board, were assembled via pre-existing template from past Cheers! events in Adobe InDesign, but resized and rearranged accordingly to fit new print dimensions. There were about 27 tags made in total; the following are 6 of them.
